Project Overview

This portfolio demonstrates the process of data preparation and cleaning uncleaned raw data using Power Query.

Download Dataset

Step-by-Step Process

  • Step 1: Download and Load Data

    • Download the dataset (Uncleaned_DS_jobs.csv)
    • Open Excel → Data → New Query → Text/CSV
    • Click Load and then Edit in Power Query

  • Step 2: Duplicate Raw Data

    • Right-click query → Select Duplicate

  • Step 3: Clean Salary Data

    • Use Text Before Delimiter on Salary Estimate
    • Create Min Salary and Max Salary columns

  • Step 4: Add Role Type Column

    • Create "Role Type" column using conditional logic based on Job Title

  • Step 5: Split Location Column

    • Use conditional logic for location corrections and split column
    • Rename and clean "State Abbreviations"

  • Step 6: Split Size Column

    • Create MinCompanySize and MaxCompanySize

  • Step 7: Handle Negative Values

    • Filter out -1 and 0 values in key columns

  • Step 8: Clean Company Names

    • Remove ratings and extra characters

  • Step 9: Copy Cleaning Steps

    • Open Advanced Editor and save code for proof

  • Step 10: Reshape and Group Data

    • Group by Role Type and Company Size, calculate average salaries

  • Step 11: Merge State Mapping

    • Merge query with state mapping workbook

  • Step 12: Group by State

    • Group by State Full Name and calculate averages

  • Step 13: View Query Dependencies

    • Use View → Dependencies to ensure all queries are linked
